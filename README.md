# Mithal: An Arabic Programming Language
Mithal is an object-oriented programming language with Java-like syntax written entirely in Classical Arabic. 

# Current Progress
Currently, the language supports basic expressions using booleans, numbers, and strings. Support for function calls, loops, and classes is currently inthe works. See `design.md` for progress and ideas on designing the language and translating keywords.

# Acknowledgements
The language and its implementation are primarily based off of Robert Nystrom's *Crafting Interpreters*, with some modifications to allow for supporting Arabic text.
